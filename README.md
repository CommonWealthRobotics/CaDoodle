# Windows install

For silent local installs

```

CaDoodle-Windows-x86_64.exe /S 

```

For silent system installs

```

CaDoodle-Windows-System-x86_64.exe /S 

CaDoodle-Windows-System-x86_64.exe /S /D=D:\CustomLocation\CaDoodle\

```
